# Dose

![Build Status](https://github.com/orbitronhd-org/dose/actions/workflows/dart.yml/badge.svg?branch=main)

Simple and modern medicine intake tracking app built using Flutter for Android.

The application allows tracking your daily medicines, keeping stock count, and reminding you through notifications and full-screen alarms.

## Features
- **Stock Tracking:** Track medicine stock and get "Low Stock" restock reminders.
- **Smart Reminders:** Notifications and full-screen alarms with auto-snooze capability.
- **Intake History:** A rich history log of taken medicines.
- **Analytics:** Visualize adherence and stock timeline using charts.
- **Data Portability:** Export and import your data easily.
